# Messenger

Мессенджер с поддержкой личных чатов, групп, каналов и музыки.

## Структура проекта

```
messenger/
├── index.html          # Главная страница (Firebase версия)
├── server.js           # Серверная часть (Node.js + Socket.io)
├── package.json        # Зависимости
├── src/                # Исходный код
├── public/             # Статика (CSS, JS, изображения)
│   ├── css/
│   └── js/
├── docs/               # Документация
└── README.md           # Описание
```

## Технологии

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Firebase (Realtime Database, Authentication)
- **Хостинг**: GitHub Pages, Firebase Hosting

## Установка

1. Клонируй репозиторий
2. Открой `index.html` в браузере
3. Замени конфиг Firebase на свой
